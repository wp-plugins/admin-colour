== Description ==

This plugin is for colour coding the admin menu. Users are able to define the colours they want in the plugins settings. I created the plugin to make it easier to switch between sections of the admin area when updating my wordpress websites. This plugin was created by Alex Wells while working as a Web Developer at Simpsons Creative 

== Installation ==
1. Upload folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
